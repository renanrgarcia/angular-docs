# Class Binding

In Angular, class binding allows you to dynamically add or remove CSS classes on an HTML element based on component properties. This can be done using the `[class.class-name]` syntax in the template or by using the `ngClass` directive.

```html
<!-- Using [class.class-name] syntax -->
<div [class.active]="isActive">This div is active: {{ isActive }}</div>
<!-- Using ngClass directive -->
<div [ngClass]="{ 'active': isActive, 'disabled': isDisabled }">
  This div is conditionally styled
</div>
```

```typescript
import { Component } from "@angular/core";
@Component({
  selector: "app-class-binding",
  templateUrl: "./class-binding.component.html",
  styleUrls: ["./class-binding.component.css"],
})
export class ClassBindingComponent {
  isActive = true;
  isDisabled = false;
}
```
