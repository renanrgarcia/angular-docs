# Host Component

In Angular, a host component is a component that contains and uses another component within its template. The host component is responsible for providing the context in which the child component operates, including passing data to it via inputs and handling events emitted by the child component.

```css
/* Styles applied to the host component element */
:host {
  display: block;
  border: 1px solid #ccc;
  padding: 16px;
  margin: 16px 0;
  border-radius: 8px;
  background-color: #f9f9f9;
}
```

```typescript
import { Component } from "@angular/core";
@Component({
  selector: "app-host-component",
  templateUrl: "./host-component.component.html",
  styleUrls: ["./host-component.component.css"],
  host: {
    class: "host-component",
    "(click)": "onHostClick()",
  },
})
export class HostComponent {
  // Component logic here
}
```

## Host Binding and Host Listener

Host binding is a feature in Angular that allows you to bind properties of the host element (the element that hosts the component) to properties of the component class. This is done using the `@HostBinding` decorator.

- It's discouraged to use `@HostBinding` or `@HostListener` (only for compatibility), instead use the `host` metadata property in the `@Component` decorator.

```typescript
import { Component, HostBinding } from "@angular/core";
@Component({
  selector: "app-example",
  templateUrl: "./example.component.html",
  styleUrls: ["./example.component.css"],
})
export class ExampleComponent {
  @HostBinding("class.active") isActive = false;
  @HostListener("click") toggleActive() {
    this.isActive = !this.isActive;
  }
}
```

## inject(ElementRef)

In Angular, `ElementRef` is a service that provides a direct reference to the DOM element associated with a component or directive. You can use `ElementRef` to access and manipulate the host element of a component.

```typescript
import { Component, inject } from "@angular/core";
import { ElementRef } from "@angular/core";
@Component({
  selector: "app-example",
  templateUrl: "./example.component.html",
  styleUrls: ["./example.component.css"],
})
export class ExampleComponent {
  private elementRef = inject(ElementRef);
  ngOnInit() {
    const nativeElement = this.elementRef.nativeElement;
    nativeElement.style.backgroundColor = "lightblue";
  }
}
```
